:: Puertas lógicas básicas:: 
Las puertas lógicas básicas, a partir de las que se pueden formar las demás son\:{
  ~%33.33333%[moodle]AND
  ~%33.33333%[moodle]OR
  ~%33.33333%[moodle]NO
  ~%-33.33333%[moodle]NOR
  ~%-33.33333%[moodle]NAND
  ~%-33.33333%[moodle]XOR
}


:: Puertas lógicas:: 
Las puertas lógicas se construyen a partir de transistores\: {TRUE}

:: Puertas lógicas 2:: 
Empareja las puertas lógicas y sus funciones\:{
  =AND -> A*B
  =OR -> A+B
  =NO -> A’
  =NOR -> A’*B’
  =NAND -> A’+B’
}

:: XOR:: 
Si quiero comparar cuando 2 bits son iguales puedo usar una puerta \:{
  =XOR
  ~AND
  ~OR
  ~NOR
  ~NAND
}

:: Forma canónica:: 
Cual de las siguientes funciones está en forma canónica\:{
  ~%50%[moodle]A
  ~%50%[moodle]A*B
  ~%-50%[moodle]A*B+C
  ~%-50%[moodle]A*B+A
}

:: Formas canónicas 2:: 
Empareja las formas canónicas\:{
  =Primera -> ABC+A’BC
  =Segunda -> (A+B+C)(A’+B+C)
}

:: NAND NOR:: 
Con qué tipo de puertas lógicas puedo realizar cualquier función lógica\: {
  ~%50%[moodle]NAND
  ~%50%[moodle]NOR
  ~%-33.33333%[moodle]AND
  ~%-33.33333%[moodle]OR
  ~%-33.33333%[moodle]XOR
}

:: Circuitos secuenciales:: 
La salida de un circuito secuencial depende solo de sus entradas\: {FALSE}
